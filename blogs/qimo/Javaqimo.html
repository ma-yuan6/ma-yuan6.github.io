# 第一次作业

  1. 分析下面程序中表达式的执行顺序，并求出最后的结果。

    public class Exercisel {
    public static void main(String[] args) {
        int x = 5;
        int i = 1, j = 20;
        int max = 0;
        if (x >= 0 && x <= 10)
            i = x * x;
        max = (x > j) ? i : j;
        System.out.print("max=" + max);
    }
}

    答案：max=20

  2. 在教材代码2-15的基础上，改造代码，打印输出以下内容：

    ***
+**
++*

    答案：

    public class HalfMatrix {
    public static void main(String[] args) {
        int lenMatrix = 3;
        printHalfMatrix(lenMatrix);
    }

    public static void printHalfMatrix(int n) {
        for (int i = 0; i < n; i++) {  // 行
            for (int j = 0; j < n; j++) {  // 每行的内容
                if (j < i) {
                    System.out.print('+');
                } else {
                    System.out.print('*');
                }
            }
            System.out.println();
        }
    }
}

  3. 编写程序，求100以内所有质数的和。

    import java.lang.Math;

public class Prime {
    public static void main(String[] args) {
        int n = 100;
        int sum = 0;
        for (int i = 0; i < n; i++) {
            sum += isPrim(i) ? i : 0;
            // if (isPrim(i)) System.out.println(i);
        }
        System.out.println("从θ到" + n + "的质数和为：" + sum);
    }

    private static boolean isPrim(int num) {
        /*
         * 判断是否是素数
         * int n: 被判断的数字
         * return: 素数返回 true，否则返回 false
         */
        boolean flag = true;
        if (num <= 1) return false;
        for (int i = 2; i <= Math.sqrt(num); i++) {  // num -> Math.sqrt(num) 为了减少循环次数
            if (num % i == 0) {
                flag = false;  // 只要能被整除就不是素数，直接跳出循环
                break;
            }
        }
        return flag;
    }
}

  4. 建立一个int数组a存储100以内的奇数。再从数组中挑选出所有的素数，将他们存储到另外 一个数组b中。要求数组b的容量与素数个数匹配，不能多也不能少。

    import java.util.ArrayList;

public class OddAndPrime {
    public static void main(String[] args) {
        int n = 100;
        int[] a = new int[(n + 1) / 2];  // 数学方式计算数组大小
        ArrayList<Integer> bList = new ArrayList<>();  // 没有办计算素数个数，又不想多循环一一遍，所以使用动态容器
        for (int i = 1, j = 0; i <= n; i += 2) {  // 获取奇数
            a[j++] = i;
        }
        for (int i : a) {
            if (isPrim(i)) bList.add(i);
        }
        Integer[] b = bList.toArray(new Integer[0]);  // 按照题目要求应该要数组
        // 分别输出结果
        for (int i : a) {
            System.out.print(i + " ");
        }
        System.out.println();
        for (int i : b) {
            System.out.print(i + " ");
        }

    }

    private static boolean isPrim(int num) {
        /*
         * 判断是否是素数
         * int n: 被判断的数字
         * return: 素数返回 true，否则返回 false
         */
        boolean flag = true;
        if (num <= 1) return false;
        for (int i = 2; i <= Math.sqrt(num); i++) {  // num -> Math.sqrt(num) 为了减少循环次数
            if (num % i == 0) {
                flag = false;  // 只要能被整除就不是素数，直接跳出循环
                break;
            }
        }
        return flag;
    }
}

  1. 导出包与导入包，参考：idea打包jar包+运行jar包亲测详解open in new window

  2. 将字符串"120,30,90,150,60"转化为double数组，使用数组服务类进行从小到大排序，使用Math类计算arr[0]的sin值

    public class Main {
    public static void main(String[] args) {
        String numberStr = "120,30,90,150,60";
        String[] numberList = numberStr.split(",");  // 按","分离
        // 使用stream流 + map
        Double[] doubles = Arrays.stream(numberList).mapToDouble(Double::parseDouble).boxed().toArray(Double[]::new);
        Arrays.sort(doubles);
        // 输出
        for (Double d : doubles) {
            System.out.println(d);
        }
        System.out.println(Math.sin(doubles[0]));
    }
}

# 第六次作业

  1. 使用代码新建目录与文件d:/files/datas/intdata..txt（P219）
  2. 在文件中分100行写入1000个整数（0-100之间的随机整数）的字符串，使用两种方法来写入（逐字节与一次性写入），并计算时间进行比较(P192)
  3. 使用按行方式读取文件内容并打印出来(P204)
  4. 删除整个目录files及目录下面的所有文件(P221)
import java.io.*;
import java.util.Random;

public class Main {
    public static void main(String[] args) {
        String directoryPath = "d:/files/datas";
        String filePath = "d:/files/datas/intdata.txt";

        // 1. 创建目录与文件
        createDirectoryAndFile(directoryPath, filePath);

        // 2. 写入文件
        writeToFile(filePath);

        // 3. 读取并打印文件内容
        readAndPrintFileContent(filePath);

        // 4. 删除目录及其下所有文件
        deleteDirectory(new File("d:/files"));
    }

    // 创建目录与文件
    public static void createDirectoryAndFile(String directoryPath, String filePath) {
        File directory = new File(directoryPath);
        if (!directory.exists()) {
            directory.mkdirs();
        }

        File file = new File(filePath);
        try {
            file.createNewFile();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

    // 写入文件
    public static void writeToFile(String filePath) {
        try {
            long startTime, endTime;
            Random random = new Random();
            BufferedWriter writer;

            // 逐字节写入
            startTime = System.nanoTime();
            writer = new BufferedWriter(new FileWriter(filePath));
            for (int i = 0; i < 100; i++) {  // 100 行
                for (int j = 0; j < 10; j++) {
                    // 一个胡数写入一次
                    writer.write(new StringBuilder().append(random.nextInt(100)).append(" ").toString());
                }
                writer.newLine();  // 换行
            }
            writer.close();
            endTime = System.nanoTime();
            System.out.println("逐字节写入时间：" + (endTime - startTime) + " 纳秒");

            // 一次性写入
            startTime = System.nanoTime();
            writer = new BufferedWriter(new FileWriter(filePath));
            StringBuilder lines = new StringBuilder();  // 先创建StringBuilder
            for (int i = 0; i < 100; i++) {
                StringBuilder line = new StringBuilder();
                for (int j = 0; j < 10; j++) {
                    line.append(random.nextInt(100)).append(" "); // 将数据放入StringBuilder中而不是写入
                }
                lines.append(line.append("\n"));
            }
            writer.write(lines.toString());  // 一次性写入全部
            writer.close();
            endTime = System.nanoTime();
            System.out.println("一次性写入时间：" + (endTime - startTime) + " 纳秒");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

    // 按行读取文件内容并打印
    public static void readAndPrintFileContent(String filePath) {
        try {
            BufferedReader reader = new BufferedReader(new FileReader(filePath));
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

    // 删除目录及其下所有文件
    public static void deleteDirectory(File file) {
        // 检查文件或目录是否存在
        if (!file.exists()) {
            System.out.println("指定的文件或目录不存在。");
            return;
        }

        // 检查如果是文件，直接删除
        if (file.isFile()) {
            file.delete();
            System.out.println("已删除文件: " + file.getAbsolutePath());
            return;
        }

        // 如果是目录，遍历其中的所有文件和子目录，并递归删除
        File[] files = file.listFiles();
        if (files != null) {
            for (File f : files) {
                deleteDirectory(f);
            }
        }

        // 删除空目录
        file.delete();
        System.out.println("已删除目录: " + file.getAbsolutePath());

    }
}
