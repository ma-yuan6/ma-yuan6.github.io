# 向量与矩阵的范数

# 向量的范数

1 范数（哈曼顿范数）

∣∣X∣∣​1​​=​i=1​∑​n​​∣x​i​​∣=∣x​1​​∣+∣x​2​​∣+⋯∣x​n​​∣

2 范数（欧几里得范数）

∣∣X∣∣​2​​=​⎷​​​​​​​i=1​∑​n​​x​i​2​​​​​=√​x​1​2​​+x​2​2​​+⋯+x​n​2​​​​​

∞ 范数

∣∣X∣∣​∞​​=​1≤i≤n​max​​{∣x​i​​∣}=max{∣x​1​​∣,∣x​2​​∣,⋯,∣x​n​​∣}

说明

  • 1 范数：向量中各个元素绝对值之和。
  • 2 范数：向量中各个元素的平方和的平方根。
  • ∞ 范数：向量中绝对值最大的元素。

# 矩阵的范数

列和范数（ L​1​​ 范数）

∣∣A∣∣​1​​=​1≤i≤n​max​​{​i=1​∑​n​​∣a​ij​​∣}

行和范数（ L​∞​​ 范数）

∣∣A∣∣​∞​​=​1≤j≤n​max​​{​j=1​∑​n​​∣a​ij​​∣}

L​2​​ 范数

∣∣A∣∣​2​​=√​λ​1​​​​​

其中，λ​1​​=max​1≤i≤n​​∣λ​i​​∣，λ​i​​ 是 A​T​​A 的特征值。

Frobenius 范数

∣∣A∣∣​F​​={​i=1​∑​n​​​j=1​∑​n​​∣a​ij​​∣​2​​}​​2​​1​​​​

说明

  • 行和范数（ L​1​​ 范数）为每一列元素绝对值之和的最大值。
  • 行和范数（ L​∞​​ 范数）为每一行元素绝对值之和的最大值。
  • L​2​​ 范数为 A​T​​A 特征值的最大值。
  • Frobenius 范数为矩阵各个元素的平方和的平方根。

# 矩阵的条件数与谱半径

Cond​p​​(A)=∣∣A∣∣​p​​∣∣A​−1​​∣∣​p​​ 为 A 的 条件数 。条件数越大，矩阵的求解误差就越大。其中 ∣∣⋅∣∣​p​​ 表示矩阵的某种范数。

谱半径 是指矩阵 A 的特征值的模的最大值。即 ρ(A)=max∣λ​i​​∣，其中 λ​i​​ 矩阵 A 的特征值。谱半径越小，矩阵A的稳定性就越高，矩阵的求解误差也就越容易受到限制。

# 代码实现

这里就不编写源码了，直接调用别人已经封装好的 API。本文使用 Python 的 numpy 库实现。

import numpy as np

# 生成一个列向量
v = np.array([1, 2, 3]).reshape(-1, 1)

# 计算向量v的1-范数、2-范数和无穷范数
print(np.linalg.norm(v, ord=1))
print(np.linalg.norm(v, ord=2))
print(np.linalg.norm(v, ord=np.inf))

# 生成一个矩阵
A = np.array([[1, 2], [3, 4]])

# 计算矩阵A的 1-范数、2-范数、无穷范数和 Frobenius-范数
print(np.linalg.norm(A, ord=1))
print(np.linalg.norm(A, ord=2))
print(np.linalg.norm(A, ord=np.inf))
print(np.linalg.norm(A, ord='fro'))

# 计算矩阵A的谱半径
print(np.max(np.abs(np.linalg.eigvals(A))))

# 计算矩阵A的条件数
print(np.linalg.cond(A, 1))
# 第二个参数指定使用的范数，默认为 2

# Jcobi 迭代

Jcobi 迭代算法是一种求解线性方程组的方法，它通过不断迭代计算，逐步逼近线性方程组的解。

他的迭代公式可以写为：X​(k+1)​​=RX​k​​+g 。其中 R 称为 迭代矩阵。

R=I−D​−1​​A，g=D​−1​​b 。

符号说明

I 为单位矩阵，D 为方程组的系数矩阵，b 为方程组等号右边组成的行列式。

假设现在有下列方程：

​⎩​⎪​⎨​⎪​⎧​​​2x​1​​−x​2​​−x​3​​=−5​x​1​​+5x​2​​−x​3​​=8​x​1​​+x​2​​+10x​3​​=11​​

现在我们可以写出方程的 迭代格式 了（也称方程的 迭代计算式 ）:

​⎩​⎪​⎨​⎪​⎧​​​x​1​(k+1)​​=0.5x​2​k​​+0.5x​3​k​​−2.5​x​2​(k+1)​​=−0.2x​1​k​​+0.2x​3​k​​+1.6​x​3​(k+1)​​=−0.1x​1​k​​−0.1x​2​k​​+1.1​​

根据上面 迭代格式 我们也可以写出 迭代矩阵R 为：

​⎝​⎛​​​0​−0.2​−0.1​​​0.5​0​−0.1​​​0.5​0.2​0​​​⎠​⎞​​

所以，对于这一题 X​k+1​​=RX​k​​+g 就为：

​⎝​⎜​⎛​​​x​1​(k+1)​​​x​2​(k+1)​​​x​3​(k+1)​​​​​⎠​⎟​⎞​​=​⎝​⎛​​​0​−0.2​−0.1​​​0.5​0​−0.1​​​0.5​0.2​0​​​⎠​⎞​​​⎝​⎛​​​x​1​k​​​x​2​k​​​x​3​k​​​​​⎠​⎞​​+​⎝​⎛​​​−2.5​1.6​1.1​​​⎠​⎞​​

最后，我们只需要取一个初始值例如：X​(0)​​=(1,1,1)​T​​，再将其带入上面计算式不断迭代，最后的结果就会无限接近准确值。

总结：

  1. 迭代方法

    • 通过迭代公式 X​k+1​​=RX​k​​+g 。

    • 通过 迭代格式 。

  2. 求迭代矩阵的方法

    • 通过公式 R=I−D​−1​​A 。

    • 通过 迭代格式 直接写出。

# Gauss-Seidel 迭代

Gauss−Seidel 迭代相对于 Jcobi 迭代主要多了将先前求得的未知变量的新值带入计算下一个未知变量。它的变量更新速度更快，因此迭代速度一般会比 Jcobi 迭代快。

他的迭代公式可以写为：X​(k+1)​​=SX​k​​+f 。其中 S 称为 迭代矩阵。

S=−(D+L)​−1​​U，f=(D+L)​−1​​b 。

符号说明

D 为方程组的系数的对角矩阵，L 为方程组的系数的下三角矩阵，

U 为方程组的系数的上三角矩阵，b 为方程组等号右边组成的行列式。

还是上面方程组，我们可以写出他的 迭代格式 ：

​⎩​⎪​⎨​⎪​⎧​​​x​1​(k+1)​​=0.5x​2​k​​+0.5x​3​k​​−2.5​x​2​(k+1)​​=−0.2x​1​(k+1)​​+0.2x​3​k​​+1.6​x​3​(k+1)​​=−0.1x​1​(k+1)​​−0.1x​2​(k+1)​​+1.1​​​(1)​(2)​(3)​​

我们可以看到（1）已经迭代出 x​1​(k+1)​​ 那么在（2）中就用其把 x​1​k​​ 替换掉。同理（3）中用 x​1​(k+1)​​ 和 x​2​(k+1)​​ 替代 x​1​k​​ 和 x​2​k​​ 。

所以，Gauss-Seidel 迭代的迭代矩阵应该这么算：

  1. 将（1）带入（2）：x​2​(k+1)​​=−0.2(0.5x​2​k​​+0.5x​3​k​​−2.5)+0.2x​3​k​​+1.6
  2. 将带入后的（2）和（1）再代入（3）

总结：

  1. 迭代方法

    • 通过迭代公式 S​k+1​​=RX​k​​+f 。

    • 通过 迭代格式 。

  2. 求迭代矩阵的方法

    • 通过公式 S=−(D+L)​−1​​U 。
    • 通过 迭代格式 不断带入得出。

# 迭代收敛的条件

对于方程组 AX=b：

  • 两者都适用的收敛条件：

    1. 迭代矩阵的 谱半径 ρ(R)=​1≤i≤n​max​​∣λ​i​​∣<1。
    2. A 为严格对角优。

  • Gauss-Seidel 迭代适用：A 正定。

严格对角优

一个矩阵被称为是“严格对角优”，当且仅当该矩阵每一行（或每一列）的绝对值最大的元素都在对角线（即该行（或该列）的主元素）上，并且每一个主元素的绝对值都大于其他非主元素的绝对值之和。

# 代码实现

Jcobi 迭代：

import numpy as np

# 定义 Jacobi 迭代函数
def jacobi_iteration(A, b, N=10, x=None):
    if x is None:
        x = np.zeros_like(b)
    D = np.diag(np.diag(A))
    R = A - D
    for i in range(N):
        x = np.dot(np.linalg.inv(D), b - np.dot(R, x))
        print("迭代 ", i+1, ": ", x)
    return x

# 设置系数矩阵 A 和常数向量 b
A = np.array([[2, -1, -1], [1, 5, -1], [1, 1, 10]])
b = np.array([-5, 8, 11])

# 调用 Jacobi 迭代函数并输出结果
x = jacobi_iteration(A, b, N=3)
print("结果: ", x)

Gauss-Seidel 迭代：

import numpy as np

# 定义 Gauss-Seidel 迭代函数
def gauss_seidel(A, b, eps=1e-10, max_iter=1000):
    n = len(b)
    x = np.zeros(n)
    for num_iter in range(max_iter):
        for i in range(n):
            s = np.dot(A[i][:i], x[:i])
            t = np.dot(A[i][i+1:], x[i+1:])
            x[i] = (b[i] - s - t) / A[i][i]
        print("迭代", num_iter+1, ":", x)
        if np.allclose(np.dot(A, x), b, atol=eps):
            break
    return x

# 设置系数矩阵 A 和常数向量 b
A = np.array([[10, -1, 0, 0], [-1, 10, -1, 0], [0, -1, 10, -1], [0, 0, -1, 10]])
b = np.array([1, 0, 1, 2])

# 调用 Gauss-Seidel 迭代函数并输出结果
x = gauss_seidel(A, b, max_iter=3)
print("结果: ", x)
