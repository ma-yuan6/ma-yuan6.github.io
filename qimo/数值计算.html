# Jcobi 迭代

Jcobi迭代算法是一种求解线性方程组的方法，它通过不断迭代计算，逐步逼近线性方程组的解。

他的迭代公式可以写为：X​(k+1)​​=RX​k​​+g 。其中 R 称为 迭代矩阵。

R=I−D​−1​​A，g=D​−1​​b 。

提示

I 为单位矩阵，D 为方程组的系数矩阵，b 为方程组等号右边组成的行列式。

假设现在有下列方程：

​⎩​⎪​⎨​⎪​⎧​​​2x​1​​−x​2​​−x​3​​=−5​x​1​​+5x​2​​−x​3​​=8​x​1​​+x​2​​+10x​3​​=11​​

现在我们可以写出方程的 迭代格式 了（也称方程的 迭代计算式 ）:

​⎩​⎪​⎨​⎪​⎧​​​x​1​(k+1)​​=0.5x​2​k​​+0.5x​3​k​​−2.5​x​2​(k+1)​​=−0.2x​1​k​​+0.2x​3​k​​+1.6​x​3​(k+1)​​=−0.1x​1​k​​−0.1x​2​k​​+1.1​​

对于一门课就学几小时的我们来说，面对题目有时会有无从下笔的感觉。因此，我们可以重点记一些专业名词并且记住他是什么样子的。比如你现在记下 迭代格式 、 迭代计算式 这两个名词。当在题目上出现这两个词，你能马上想到它是类似上面这样的一个方程组。马上就会有思路。

照这样复习的思路来，花几个小时拿一个还能不错分数其实不是很难。当然，这仅限于临时抱佛脚应付考试，并且在题目不难的情况下。

提醒

我会将所有专业名词标为斜体。

根据上面 迭代格式 我们也可以写出 迭代矩阵R 为：

​⎝​⎛​​​0​−0.2​−0.1​​​0.5​0​−0.1​​​0.5​0.2​0​​​⎠​⎞​​

所以，对于这一题 X​k+1​​=RX​k​​+g 就为：

​⎝​⎜​⎛​​​x​1​(k+1)​​​x​2​(k+1)​​​x​3​(k+1)​​​​​⎠​⎟​⎞​​=​⎝​⎛​​​0​−0.2​−0.1​​​0.5​0​−0.1​​​0.5​0.2​0​​​⎠​⎞​​​⎝​⎛​​​x​1​k​​​x​2​k​​​x​3​k​​​​​⎠​⎞​​+​⎝​⎛​​​−2.5​1.6​1.1​​​⎠​⎞​​

最后，我们只需要取一个初始值例如：X​(0)​​=(1,1,1)​T​​，再将其带入上面计算式不断迭代，最后的结果就会无限接近准确值。

总结：

  1. 迭代方法

    • 通过迭代公式 X​k+1​​=RX​k​​+g 。

    • 通过 迭代格式 。

  2. 求迭代矩阵的方法

    • 通过公式 R=I−D​−1​​A 。

    • 通过 迭代格式 直接写出。

# Gauss-Seidel 迭代

Gauss-Seidel 迭代相对于 Jcobi 迭代主要多了将先前求得的未知变量的新值带入计算下一个未知变量。它的变量更新速度更快，因此迭代速度一般会比 Jcobi 迭代快。

他的迭代公式可以写为：X​(k+1)​​=SX​k​​+f 。其中 S 称为 迭代矩阵。

S=−(D+L)​−1​​U，f=(D+L)​−1​​b 。

提示

D 为方程组的系数的对角矩阵，L 为方程组的系数的下三角矩阵，

U 为方程组的系数的上三角矩阵，b 为方程组等号右边组成的行列式。

还是上面方程组，我们可以写出他的 迭代格式 ：

​⎩​⎪​⎨​⎪​⎧​​​x​1​(k+1)​​=0.5x​2​k​​+0.5x​3​k​​−2.5​x​2​(k+1)​​=−0.2x​1​(k+1)​​+0.2x​3​k​​+1.6​x​3​(k+1)​​=−0.1x​1​(k+1)​​−0.1x​2​(k+1)​​+1.1​​​(1)​(2)​(3)​​

我们可以看到（1）已经迭代出 x​1​(k+1)​​ 那么在（2）中就用其把 x​1​k​​ 替换掉。同理（3）中用 x​1​(k+1)​​ 和 x​2​(k+1)​​ 替代 x​1​k​​ 和 x​2​k​​ 。

所以，Gauss-Seidel 迭代的迭代矩阵应该这么算：

  1. 将（1）带入（2）：x​2​(k+1)​​=−0.2(0.5x​2​k​​+0.5x​3​k​​−2.5)+0.2x​3​k​​+1.6
  2. 将带入后的（2）和（1）再代入（3）

总结：

  1. 迭代方法

    • 通过迭代公式 S​k+1​​=RX​k​​+f 。

    • 通过 迭代格式 。

  2. 求迭代矩阵的方法

    • 通过公式 S=−(D+L)​−1​​U 。
    • 通过 迭代格式 不断带入得出。

# 迭代收敛的条件

对于方程组 AX=b：

  • 两者都适用的收敛条件：

    1. 迭代矩阵_ 的谱it 半径 ρ(R)=​1≤i≤n​max​​∣λ​i​​∣<1。
    2. A 为严格对角优。

  • Gauss-Seidel 迭代适用：A 正定。

严格对角优

一个矩阵被称为是“严格对角优”，当且仅当该矩阵每一行（或每一列）的绝对值最大的元素都在对角线（即该行（或该列）的主元素）上，并且每一个主元素的绝对值都大于其他非主元素的绝对值之和。

# 代码实现

Jcobi 迭代：

import numpy as np

# 定义 Jacobi 迭代函数
def jacobi_iteration(A, b, N=10, x=None):
    if x is None:
        x = np.zeros_like(b)
    D = np.diag(np.diag(A))
    R = A - D
    for i in range(N):
        x = np.dot(np.linalg.inv(D), b - np.dot(R, x))
        print("迭代 ", i+1, ": ", x)
    return x

# 设置系数矩阵 A 和常数向量 b
A = np.array([[2, -1, -1], [1, 5, -1], [1, 1, 10]])
b = np.array([-5, 8, 11])

# 调用 Jacobi 迭代函数并输出结果
x = jacobi_iteration(A, b, N=3)
print("结果: ", x)

Gauss-Seidel 迭代：

import numpy as np

# 定义 Gauss-Seidel 迭代函数
def gauss_seidel(A, b, eps=1e-10, max_iter=1000):
    n = len(b)
    x = np.zeros(n)
    for num_iter in range(max_iter):
        for i in range(n):
            s = np.dot(A[i][:i], x[:i])
            t = np.dot(A[i][i+1:], x[i+1:])
            x[i] = (b[i] - s - t) / A[i][i]
        print("迭代", num_iter+1, ":", x)
        if np.allclose(np.dot(A, x), b, atol=eps):
            break
    return x

# 设置系数矩阵 A 和常数向量 b
A = np.array([[10, -1, 0, 0], [-1, 10, -1, 0], [0, -1, 10, -1], [0, 0, -1, 10]])
b = np.array([1, 0, 1, 2])

# 调用 Gauss-Seidel 迭代函数并输出结果
x = gauss_seidel(A, b, max_iter=3)
print("结果: ", x)
